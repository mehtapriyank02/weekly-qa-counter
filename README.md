# Weekly QA Counter

A small GitHub Pages + Supabase dashboard for weekly QA tracking.

## Files
- `index.html`
- `styles.css`
- `app.js`
- `config.js`

## Setup
1. In Supabase, go to Project Settings → API.
2. Copy:
   - Project URL
   - anon public key
3. Paste both inside `config.js`.
4. In Supabase, go to Authentication → Users.
5. Add users for Priyank and Parth with email/password.
6. Upload these files to a GitHub repo.
7. Turn on GitHub Pages from Settings → Pages.

Do not paste or expose your Supabase service role key.
Only use the anon public key.
